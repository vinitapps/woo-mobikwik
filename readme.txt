This is the unofficial MobiKwik payment gateway wallet plugin for WooCommerce. Allows you to accept payments via MobiKwik wallet.

== Description ==

This is the unofficial MobiKwik payment gatewayplugin for WooCommerce. Allows you to accept mobikwik wallet payments with the WooCommerce plugin. Works across all browsers, and is compatible with the latest WooCommerce.

This is compatible with WooCommerce>=3.3.It has been tested upto the 3.1.1 WooCommerce release.

== DEPENDENCIES ==

WordPress v3.9.2 and later
Woocommerce v2.4 and later
PHP v5.6.0 and later
php-curl extension
